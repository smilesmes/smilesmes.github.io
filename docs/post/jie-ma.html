前言

这篇内容主要分享几个可免费使用的 VOIP 短信接码网站，并附带简单评价，方便有需要的人自行筛选使用。

另外，如果你想先查询一个号码是不是虚拟号，可以使用下面这个工具：

  • 号码类型查询：https://www.freecarrierlookup.com/

说明：

  • 如果查询结果里 Is Wireless: n，通常表示它是虚拟号。
  • 临时公用号码存在较高风险，请勿用于绑定重要账号。

1. sandbox httpSMS

  • 官网：https://sandbox.httpsms.com/

简介：
可以免费使用美国号码发送和接收测试短信。

评价：

  • 无需注册
  • 不定期更换实体美国号码
  • 每天可发送 10 条测试短信
  • 接收短信无限制

适合场景：

  • 简单测试短信收发
  • 临时验证非重要服务

2. Twilio

  • 注册页：https://login.twilio.com/u/signup

简介：
Twilio 提供编程式短信服务，支持大量虚拟号码，适合开发和 API 测试类需求。

评价：

  • 需要实体号码接码验证，才能注册免费试用
  • 免费试用期通常为一年
  • 提供大量虚拟号码
  • 支持 API 短信能力
  • 对验证码短信有一定过滤
  • 免费试用限制较多
  • 可通过邮件申请 port out

适合场景：

  • 开发者测试短信 API
  • 有编程需求的用户
  • 需要长期研究短信能力的平台用户

3. eSIM Plus

  • 地址：https://esimplus.me/temporary-numbers

简介：
提供多种临时虚拟号码。

评价：

  • 各种虚拟号码可选
  • 适合自行筛选和尝试

适合场景：

  • 查找不同地区的临时号码
  • 多号码尝试需求

4. ZUSMS

  • 官网：https://www.zusms.com/

简介：
免费在线短信接收平台，提供美国、英国、香港、加拿大等全球临时手机号，可用于接收验证码。

评价：

  • 无需注册即可使用
  • 提供多个国家和地区号码
  • 部分号码无效，需要自行筛选最近有更新的号码
  • 有 2 个香港号码可用

适合场景：

  • 临时接收验证码
  • 需要香港或海外号码做简单测试

5. Fongo

简介：
加拿大虚拟号服务。

评价：

  • 需要在 Google Play 商店切区下载
  • 世界版无限制，但收费 5 美元
  • 验证码接收功能收费

适合场景：

  • 加拿大虚拟号需求
  • 愿意付费获取更多功能的用户

使用建议

在使用这类临时接码平台时，建议注意以下几点：

  1. 不要绑定重要账号
    包括邮箱、支付、社交主号等核心账户。

  2. 优先测试活跃号码
    尽量选择最近仍有更新、能正常接收短信的号码。

  3. 明确用途
    这类号码更适合注册测试、临时验证和开发调试，不适合长期稳定使用。

  4. 注意平台限制
    某些平台虽然免费，但会限制发送条数、接收能力或注册方式。

总结

如果只是想找一个临时接码平台，sandbox httpSMS 和 ZUSMS 相对更适合直接上手。
如果你是开发者，想测试短信 API，Twilio 会更有研究价值。
如果你需要更多类型的虚拟号码，也可以看看 eSIM Plus 和 Fongo。

不过还是那句话：临时公用号码只能用于低风险场景，切勿绑定重要账号。
